# Motor-de-deteccion-de-fallas

Motor configurable para la deteccion temprana de fallas en sistemas monitoreados, basado en reglas flexibles

# Incluye

Generacion de eventos de alertas

Reglas configurables

Ventanas deslizantes de datos

Dashboard en locahost

# Uso

-Iniciar el programa con: streamlit run dashboard.py

-Visualizar Dashboard en localhost

-Descargar Reporte de datos

# Casos de uso

Maquinas industriales
Sensores
Analisis preventivo
